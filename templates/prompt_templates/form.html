Template Guidance

            

            

                
Prompt templates control how comments are generated:

                
    
                    
  • System Prompt: Defines behaviour, tone, and restrictions.
    • 
                    
  • User Prompt: Provides article-specific context and instructions.
    • 
                    
  • Comments are automatically prefixed when posted to myMoment.
    • 
                

                

                
Supported placeholders:

                
    
                    
  • {article_title} * - headline of the article
    • 
                    
  • {article_author} * - displayed author name
    • 
                    
  • {article_content} * - full text content
    • 
                    
  • {article_raw_html} - raw HTML content
    • 
                

                
* = always available

            

        


        

            

                
Best Practices

            

            

                
    
                    
  • Keep instructions concise and explicit.
    • 
                    
  • Remind the AI to respect platform guidelines.
    • 
                    
  • Use placeholders instead of hard-coding article details.
    • 
                    
  • Encourage varied tone to avoid repetitive comments.
    • 
                    
  • Validate changes with the preview before saving.
